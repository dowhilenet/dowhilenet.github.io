网页控件（UIWebView）加载本地数据，文件

  1. 1. 我们用上述三种方法来加载一个HTML文件
    1. 1.1. 使用loadHTMLString方法加载HTML内容
  • 欢迎来到：百度
    1. 0.1. 使用loadRequest方法加载本地资源（也可用于加载服务器资源）
    2. 0.2. 先将内容保存成NSData数据，再使用loadData方法加载

    • 使用UIWebView加载本地数据或资源有如下三种方式：

    1，使用loadHTMLString方法加载HTML内容

    2，使用loadRequest方法加载本地资源（也可用于加载服务器资源）

    3，先将内容保存成NSData数据，再使用loadData方法加载

    我们用上述三种方法来加载一个HTML文件

    使用loadHTMLString方法加载HTML内容

    ```swift
    webVieew.loadHTMLString(“

    欢迎来到：百度

    “, baseURL: nil)
    ```

    使用loadRequest方法加载本地资源（也可用于加载服务器资源）

    ```swift
    let path = NSBundle.mainBundle().pathForResource(“index”, ofType: “html”)
    let pathNSURL = NSURL(fileURLWithPath: path!)
    let request = NSURLRequest(URL: pathNSURL)
    webVieew.loadRequest(request)
    ```

    先将内容保存成NSData数据，再使用loadData方法加载

    ```swift
    // 首先呢要把这个HTML文件加入到项目之中

    class WebViewController: UIViewController {

    override func viewDidLoad() {
    super.viewDidLoad()
    //检查 BullsEye.html是否存在
    if let htmlFile = NSBundle.mainBundle().pathForResource("BullsEye", ofType: "html"){

        let htmlData = NSData(contentsOfFile: htmlFile)

        let baseURL = NSURL.fileURLWithPath(NSBundle.mainBundle().bundlePath)

        web.loadData(htmlData, MIMEType: "text/html", textEncodingName: "UTF-8", baseURL: baseURL)
    }
}


//UIWebView
@IBOutlet weak var web: UIWebView!

    }
    ```
